LEO Core API deployed successfully!

To access the service:

  export POD=$(kubectl get pods -l app={{ include "leo-core.name" . }} -o jsonpath='{.items[0].metadata.name}')
  kubectl port-forward $POD 8000:8000

Then open in browser:
  http://localhost:8000/healthz

For MCP server:
  kubectl port-forward deployment/leo-core-mcp 8800:8800
